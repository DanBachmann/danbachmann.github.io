ECS7022P: Computational Creativity Project

Dataset

An additional value added component was a creation of dataset consisting of nearly 10,000 image sets rendered in contolled lighting with a neutral grey background. For each image there is the original 512x512 pixel portrait, a depth map, a painted face and the image used to create the face paint. The depth map was not used in this project; however, it would be useful for other training considering facial geometry.
