ECS7022P: Computational Creativity Project

Face pAInt

The aim of this project was to create/assist with generative fine-art using computational photography techniques to extend a traditional studio shoot. This is achieved by training a diffusion model to understand the concept that an image/pattern which typically would appear as a background can be applied onto a face. For this to work, model needs to learn the 3d semantic map of the face.
Images are the artefacts which can be a upscaled to high resolution as a final image or used as layers on top of the original image to maintain fine details with the options of further post processing. The goal is an aesthetic result intended for artistic display.
The motivation for this project was to create vibrant portraits using the look of paint on a face. Inspiration had come from images on-line; however, many of the most vibrant ones were clearly AI. I wanted to do this using real people. It reminded me of a photo shoot in the past with a make-up artist experimenting. I didn’t know what would come out of the changing room. In post production I could do more, but I was limited to working on a 2d plane, where a true make-up artist worked on a 3d face.

This system touches on all of these points: intense face painting images, multiple random variations on a single pose and near pixel bound output so the output can be pulled into layers for further artistic expression maintaining the 3d feeling. The best post-production steps could be bright into the output pipeline later.
